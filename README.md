# Mesh-Quality
A C++ module for evaluating the quality of 2D meshes used in structural Dynamics

## How to build
1. make a build directory
2. change into the directory
3. Run cmake
4. Run make (on linux)

## Dependencies
* GMSH 4.4.1 or higher
* CGAL 5.0 or higher
* boost for CGAL
* C++17 or newer

## Tested Compilers
* clang 9.0.0-2
* g++ 9.2.1
